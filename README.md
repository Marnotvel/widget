# Проект "Мой проект Python"
## Описание:

Этот проект - виджет для работы с банковскими операциями клиента

## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/Marnotvel/widget
```
2. Установите зависимости:

```
pip install -r requirements.txt
```

## Использование

Проект направлен на работу с банковскими операциями. В будущем в проект будут добавлены новые функции
